You can type $\LaTeX$ here and it will be rendered on the right. Enclose in \$...\$ for inline or \$\$...\$\$ for block. Type CTRL-M to bring up a LaTeX search helper. You can also use markdown syntax.

Using the LaTeX Keyword Search

To make writing math in LaTex easier, we've implemented a search feature. To use it, press CTRL-M, and use the arrow keys (â‡…) to navigate.

Press ENTER to insert the selected syntax at the cursor position, or ESC to close the search bar.
